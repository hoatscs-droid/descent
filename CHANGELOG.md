# Changelog

All notable changes to Descent are documented here.

## 2026-07-09 — Current release

### Added

- Intention-led Sit library with Clarity, Daytime, and Deep Sleep groups
- Expanded Tinker engine with two oscillators, configurable noise, pulse controls, and precision input
- Live binaural-difference and approximate brainwave-band visualization
- Local custom-preset management
- Redesigned identity, reactive resonance orb, and unified color progression
- Continuous Explore descent visualization
- Local icon assets, install metadata, and offline application-shell caching
- Release documentation, licensing, privacy, and scientific framing

### Changed

- Refined spacing, hierarchy, responsive behavior, and selected states throughout the interface
- Reworked Explore from isolated cards into a continuous progression
- Consolidated and clarified reset controls
- Improved information and practice guidance

### Notes

- No accounts, analytics, advertising, or remote APIs
- Saved presets and preferences remain local to the browser
- Binaural-beat frequency bands are presented as orientation, not guaranteed brain-state induction
